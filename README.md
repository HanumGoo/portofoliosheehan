# Sheehan Andya Portfolio

Static portfolio website for software engineering job applications.

## Maintainability

- Edit portfolio content in `script.js`.
- Edit page structure in `index.html`.
- Edit the visual system and responsive layout in `style.css`.
- Keep images and certificates in `folder experience and sertifikat/`.
- Keep the resume PDF in the project root so the existing resume button works.

## Local Preview

```bash
python -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
